Welcome to my Home Labs GitHub Space, a dedicated corner of the internet where I document my journey as an aspiring Cybersecurity/SOC Analyst. This repository serves as a living portfolio of hands-on labs, experiments, and technical write-ups that I've built and explored in my own home lab environment.

Cybersecurity is a field that demands more than theoretical knowledge. It requires practice, curiosity, and a willingness to break things in order to understand how they work. Every lab documented here reflects that mindset. Whether it's spinning up vulnerable machines, analyzing network traffic, simulating attacks, or building detection rules, each project is a step forward in sharpening the skills needed to operate effectively in a Security Operations Center.

My goal is to use this space to not only track my own growth, but to create resources that are clear, reproducible, and useful to others who are walking a similar path.
This is a continuous work in progress, and it grows with every lab I complete. Feel free to explore, learn, and reach out. Let's build together.
